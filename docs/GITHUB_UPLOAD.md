# GitHub Upload Guide

1. Create a new empty repository on GitHub, for example `kabelwerkstatt`.
2. Extract this ZIP file.
3. Open a terminal inside the extracted project folder.
4. Run:

```bash
git init
git add .
git commit -m "Initial release of KabelWerkstatt"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/kabelwerkstatt.git
git push -u origin main
```
